Setup

The setup of Conditioner is a two step process.

Conditioner needs an AMD loader to function. Currently RequireJS is a great choice. You can get it at http://requirejs.org, if you haven't played around with RequireJS before, follow the Getting Started guide for a quick meet and greet.

Now you've got a feel for RequireJS it is time to setup Conditioner.

For now we'll use RequireJS as an example, first it needs to know where to find the Conditioner framework. You'll need to add a map entry to the RequireJS configuration object pointing to the location of Conditioner.
